## ✨ Features

| Feature | Description |
|---|---|
| ➕ Add Tasks | Quickly add tasks with a single input |
| ✅ Complete / Undo | Toggle tasks between done and active |
| 🗑️ Delete | Remove tasks instantly |
| 📅 Due Dates | Assign deadlines; overdue tasks are highlighted |
| 📁 Categories | Organize by Work, Personal, Shopping, or Health |
| 🔴 Priority Levels | Set Low, Medium, or High priority per task |
| 🔍 Search | Instantly filter tasks as you type |
| 📊 Progress Bar | Visual completion percentage tracker |
| 🌙 Dark / Light Mode | System-aware theme with manual toggle |
| 💾 Auto Save | All tasks persisted via `localStorage` |
| 📱 Responsive | Works seamlessly on mobile and desktop |

```bash
cd TO-DO-List
# Open index.html in any browser
```

## 🗂️ How It Works

1. **Add a task** — Type in the input, optionally set a due date, category, and priority, then hit `+`
2. **Filter tasks** — Use the status tabs (All / Active / Done) and category chips to narrow your view
3. **Search** — The search bar filters tasks in real time
4. **Track progress** — The progress bar updates automatically as tasks are completed
5. **Switch themes** — Click the toggle in the header to switch between light and dark mode
6. **Data persists** — Everything is saved to `localStorage`, so your tasks survive page refreshes
